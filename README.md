# نظام إخلاء الطرف الإلكتروني
## مستشفى ضباء العام - منطقة تبوك

---

## نظرة عامة
نظام إلكتروني متكامل لإدارة عملية إخلاء الطرف في المستشفيات، يهدف إلى:
- تقليل مدة إخلاء الطرف إلى أقل من 24 ساعة
- تقليل العبء الإداري على الأقسام
- تحقيق التحول الرقمي ورفع كفاءة الأداء
- تحسين تجربة الموظف داخل المنشأة

---

## المتطلبات
- **Node.js** (الإصدار 18 أو أحدث) - [تحميل](https://nodejs.org/)
- متصفح ويب حديث (Chrome, Edge, Firefox)

---

## التثبيت

### 1. تثبيت Node.js
قم بتحميل وتثبيت Node.js من الرابط أعلاه.

### 2. تثبيت المكتبات
```bash
npm install
```

### 3. تشغيل النظام
```bash
npm start
```

أو للتطوير مع التحديث التلقائي:
```bash
npm run dev
```

### 4. فتح المتصفح
افتح المتصفح على: `http://localhost:3000`

## المميزات

### 1. النموذج الإلكتروني
- إنشاء طلب إخلاء طرف جديد
- تعبئة بيانات الموظف تلقائياً
- اختيار سبب إخلاء الطرف
- إقرار بصحة البيانات

### 2. تتبع الطلبات
- متابعة حالة الطلب في الوقت الفعلي
- عرض خطوات إخلاء الطرف لكل إدارة
- إشعارات فورية عند تحديث الحالة

### 3. لوحة التحكم
- إحصائيات شاملة عن الطلبات
- عرض الطلبات حسب الحالة والقسم
- مؤشرات الأداء الرئيسية
- تصدير البيانات إلى Excel

### 4. الإشعارات البريدية
- إشعار تلقائي لكل إدارة عند إنشاء طلب جديد
- تحديثات عند الموافقة أو الرفض
- إمكانية إعداد SMTP خاص بالمؤسسة

### 5. التوقيع الإلكتروني
- دعم التوقيع الرقمي
- سجل تدقيق كامل للعمليات

## مؤشرات النجاح
- تقليل مدة إخلاء الطرف إلى أقل من 24 ساعة
- ارتفاع نسبة الطلبات المنجزة إلكترونياً
- انخفاض الأخطاء أو فقدان النماذج
- ارتفاع مستوى رضا الموظفين

---

## التطوير المستقبلي
- ربط النظام مع نظام الموارد البشرية
- إضافة خاصية التوقيع الإلكتروني المعتمد
- إنشاء تطبيق داخلي لخدمات الموظفين
- التعميم على مستوى التجمع الصحي

---

## إعداد البريد الإلكتروني
لتفعيل الإشعارات البريدية، قم بتعديل ملف `.env`:

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
```

> **ملاحظة:** لاستخدام Gmail، قم بإنشاء كلمة مرور خاصة بالتطبيقات من إعدادات حساب Google.
